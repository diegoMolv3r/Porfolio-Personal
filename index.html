SOBRE MÍ

Soy estudiante universitario de Desarrollo Web en la Universidad Nacional de La Matanza (UNLaM), donde actualmente tengo 13 de las 20 materias de la carrera aprobadas. Una vez finalizada esta tecnicatura, planeo continuar mis estudios en la misma institución cursando la Licenciatura en Gestión de Tecnologías de la Información, con el objetivo de seguir ampliando mi formación y fortalecer mi perfil profesional.

En el desarrollo Front-end, tengo sólidos conocimientos en HTML, CSS y JavaScript, que me permiten construir interfaces modernas y accesibles. Utilizo frameworks como Bootstrap 5 y Tailwind CSS para lograr diseños eficientes y responsivos. Además, tengo experiencia con jQuery y AJAX para la manipulación dinámica del DOM y la integración de funcionalidades asincrónicas en aplicaciones web.

En el ámbito del Back-end, desarrollo aplicaciones utilizando Java y PHP, aplicando el patrón Modelo-Vista-Controlador (MVC) para construir sistemas escalables y organizados. Trabajo con Spring MVC en Java, siguiendo buenas prácticas de arquitectura. Manejo bases de datos relacionales con MySQL, integrándolas mediante Hibernate y JPA en Java para facilitar el mapeo objeto-relacional, y aplicándolas también en proyectos PHP. Además, elaboro Diagramas Entidad-Relación para diseñar estructuras normalizadas, eliminar redundancia y mejorar el rendimiento.

Para asegurar la calidad del software, aplico la metodología Test Driven Development (TDD), desarrollando primero las pruebas automatizadas. En el entorno JavaScript utilizo Jasmine, y en Java, trabajo con JUnit 5, junto con Hamcrest para aserciones y Mockito para simulación de dependencias. Estas herramientas me permiten garantizar código confiable y mantenible.

Manejo herramientas clave para el desarrollo moderno como Postman para pruebas de APIs, Git para control de versiones, Docker para entornos aislados, y Maven para automatizar la gestión de proyectos Java.

Mi objetivo es seguir creciendo en el mundo del desarrollo web, aprendiendo y aplicando nuevas tecnologías para ofrecer soluciones innovadoras, eficientes y centradas en la experiencia del usuario.
