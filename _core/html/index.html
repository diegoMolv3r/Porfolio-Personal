SOBRE MI

Soy estudiante universitario de desarrollo web en la Universidad Nacional de La Matanza (UNLaM), donde he adquirido sólidos conocimientos en HTML, CSS y JavaScript para desarrollo Front-end. Además, tengo formación en diseño gráfico, lo que me permite crear interfaces web atractivas y funcionales.


Para el Back-end tengo una sólida base en Java, con enfoque en programación orientada a objetos, abarcando conceptos clave como polimorfismo, clases abstractas, interfaces, herencia y pruebas unitarias con JUNIT4 aplicando la metodología TDD. Teniendo en cuenta estos conocimientos, me estoy introduciendo en TypeScript para luego complementar con un framework como Angular.


Por otro lado, tengo conocimientos en MySQL, tanto en el uso de Workbench como en Command Line Client, lo que me permite realizar consultas complejas. También cuento con conocimientos sobre bases de datos relacionales y en la elaboración de Diagramas Entidad-Relación, buscando las mejores opciones para eliminar la redundancia y obtener un mejor rendimiento.


Mi objetivo es seguir creciendo en el mundo del desarrollo web, aprendiendo y aplicando nuevas tecnologías para ofrecer soluciones innovadoras y eficientes.
